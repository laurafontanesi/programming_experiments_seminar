"WPA 1 answers"

To change the number of trials from 10 to 5, you should change the trials attribute:

<block testlt>
/trials = [1-15=noreplace(testlt)]
/bgstim = (leftresp, rightresp)
/preinstructions = (lotterytest)
</block>


To change the trial items:

<item rpraclt>
/1 = "lt1.png"
/2 = "lt2.png"
/3 = "lt3.png"
/4 = "lt4.png"
/5 = "prlt7.png"
/6 = "prlt8.png"
</item>


To change the response keys:

<trial rpraclt>
 /stimulustimes = [0=Ready; 600=rpraclt]
 /iscorrectresponse = [trial.rpraclt.latency>300 && trial.rpraclt.latency<20000 && (trial.rpraclt.response == 31 || trial.rpraclt.response == 37)]
 /validresponse = ("s", "k")
 /errormessage = true(speed, 2000)
 /pretrialpause = values.tpause
 /posttrialpause = values.ppause
 </trial>

<trial tpracltleft>
 /stimulustimes = [0=Ready; 600=tpracltleft]
 /correctresponse = ("s")
 /validresponse = ("s", "k")
 /errormessage = true(lefterror, 2000)
/ ontrialend = [if (trial.tpracltleft.correct == false) values.understood = values.understood +1]
 /pretrialpause = values.tpause
 /posttrialpause = values.ppause
 </trial>

<trial tpracltright>
 /stimulustimes = [0=Ready; 600=tpracltright]
 /correctresponse = ("k")
 /validresponse = ("s", "k")
 /errormessage = true(righterror, 2000)
/ ontrialend = [if (trial.tpracltright.correct == false) values.understood = values.understood +1]
 /pretrialpause = values.tpause
 /posttrialpause = values.ppause
 </trial>

<trial testlt>
 /stimulustimes = [0=Ready; 600=testlt]
 /iscorrectresponse = [trial.testlt.latency>300 &&trial.testlt.latency<20000 && (trial.testlt.response == 31 || trial.testlt.response == 37)]
 /validresponse = ("s", "k")
 /errormessage = true(speed, 2000)
 /pretrialpause = values.tpause
 /posttrialpause = values.ppause
 </trial>

Note that you should also change the keyboard codes (go to Tools>keyboard Scancodes in the Inquisit menu) as well as the valid responses.

To change the instructions, you can canghe any of the lottery instruction pages.
